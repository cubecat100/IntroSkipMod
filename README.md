# IntroSkip

# English

This mod shortens the startup logo intro in Slay the Spire 2 and displays the main menu more quickly.

## Features

- Loads the essential main menu resources in parallel while the logo animation is playing.
- Shows loading progress as both a percentage and a progress bar during the animation.
- Continues loading deferred startup resources after entering the main menu.

# Korea

Slay the Spire 2의 시작 로고 인트로 구간을 줄이고 메인 메뉴를 더 빠르게 표시하도록 만드는 모드입니다.

## 기능

- 로고 애니메이션이 재생되는 동안 메인 메뉴 필수 리소스 로딩을 병렬로 실행합니다.
- 애니메이션 진행 중 로딩 진행률을 퍼센트와 진행 바로 표시합니다.
- 메인 메뉴 진입 후 지연 로딩 대상 시작 리소스를 이어서 불러옵니다.
